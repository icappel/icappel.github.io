About Me

This page is a detailed summary of my skills, education, and experience. For a more succinct summary, see my résumé.

Skills

While most of the skills listed below are framed in relation to web development, my computer science education and programming experience make me a suitable candidate for any entry-level software development position.

General Development Skills

I have used Git and GNU/Linux extensively, and I am proficient in standard GNU/Linux command line tools and BASH scripting. I have a solid grasp of standard data structures and algorithms. I am familiar with object-oriented design and programming and have used these extensively. Finally, I am familiar with agile development methodologies.

Front-End Skills

In addition to a strong command of the web fundamentals (HTML, CSS, and CSS), I have experience writing websites and web applications using frameworks and other tools such as React, SCSS, TypeScript, and Bootstrap. Responsive design is second-nature to me, and I am familiar with the theory and practice of making accessible websites.

Back-End Skills

I have implemented web application backends using Node.js and the Express server framework. I have implemented a MongoDB database as the backend for a web app using Mongoose for object modelling. Additionally, I wrote significant amounts of SQL, Python, C, and Java code during my time in university. I also have some familiarity with Docker from my internship.

Education

I graduated from the University of Toronto in June 2020. I obtained an Honours Bachelor of Science with High Distinction, with a double-major in Computer Science and Philosophy. My GPA was 3.82, and I was given the distinction of Dean’s List Scholar.

My computer science coursework included topics such as data structures, databases, software design, algorithm design and analysis, operating systems, and web development. In my philosophy major I focused on formal logic, ethics, and the philosophy of mind.

During my time at university I was awarded the following scholarships and awards:

  • St. Michael's College Silver Medal
  • Lawrence E. Lynch In-Course Scholarship in Philosophy
  • St. Michael's College / C. L. Burton Trust Fund In-Course Scholarship
  • President's Entrance Scholarship

Experience

From September 2018 to August 2019 I worked as an Engineering Intern at Sigma Systems (now Hansen Technologies) as part of the University of Toronto’s Professional Experience Year program.

The bulk of my time here was spent working on a front-end web application written using React, MobX, TypeScript, and Bootstrap 3. I wrote real code set to go into production; this was not an intern project. I worked under a more experienced developer who delegated features and other tasks such as POCs which I then implemented on my own. At one point I proposed and implemented a refactor which saved over 1600 lines of code and improved maintainability and extensibility.

Before that, I created an internal E-Learning course about how Docker works to help bridge an understanding gap between technical and non-technical employees.

Some additional experiences I had working at Sigma include:

  • completing courses in information security and secure development practices
  • working with enterprise-level software development practices (e.g. using a more advanced source control system involving Git, Gerrit, and Jenkins)
  • working on an agile team with daily standups
